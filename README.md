# Spotify Clone

A Spotify-inspired music interface developed with React Native and Expo.

## Features
- Home screen with music recommendations
- Song detail page
- Navigation between screens
- Spotify-like dark interface
- Turkish music examples

## Technologies
- React Native
- Expo
- Expo Router
- JavaScript

## Project Structure
- components/
- screens/
- src/app/
